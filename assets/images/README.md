# Logo Image

Place your logo image file here with the name: `logo.png`

**Requirements:**
- File name: `logo.png`
- Recommended size: 200x200 pixels or larger (square format works best)
- Supported formats: PNG (recommended), JPG, or WebP

The logo will be displayed on:
- Login Selection Page (centered at top)
- Student Login Page (centered at top)
- Teacher Login Page (centered at top)
- Registration Page (centered at top)

If the logo file is not found, a fallback icon will be displayed instead.
